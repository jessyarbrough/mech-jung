About Mechanical Jung

Mechanical Jung is an online personality profiler that draws from the writings of famed Psychologist Carl Jung. Using his eight personality types, Mechanical Jung takes text input, either user entered or from their Twitter posts, and returns a Jungian personality profile indictative of the text they've entered.

Personality tests are time consuming and unspecific. Mechanical Jung is instant and allows the user to see how their personality changes across different mediums, whether its professional writing or social media, giving them a robust view of how they communicate. We built this application as a CSCE 470 project at Texas A&M University taught by Dr. James Caverlee. It utilizes many core concepts we received exposure to in class (classifiers, web APIs, web crawling, ranked retrieval, TFIDF, etc.) and combines them into a simple application with powerful results.

We've gathered thousands of quotes and tweets from known personality types of famous and historical individuals. Using our database as training data, we created a Support Vector Classifier that uses users input to predict a Jungian Personality type.
