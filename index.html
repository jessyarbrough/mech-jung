About Mechanical Jung

Mechanical Jung is an online personality predictor based on the theories of psychologist Carl Jung, similar to the contemporary Myers-Briggs Type Indicator (MBTI). Instead of answering questions, the user enters some text or a Twitter handle and Mechanical Jung uses advanced text classification techniques to predict the personality type of the author or Twitter user. You can try it here.

Mechanical Jung began as a semester project in CSCE 470 Information Storage and Retrieval at Texas A&M University, taught by Dr. James Caverlee. Mech Jung uses most of the course's core concepts: vector space modeling (of text), stopwords and stemming, TF-IDF, search engine optimization (Hover + Google Webmaster Tools FTW guys), kNN and naive Bayes classification, crawling, and interface design (like our web 1.0 homepage?).

Naturally, we took it a step further, using natural language processing to normalize text and, in some cases, filter based on part-of-speech and lemmatize words. We even used support vector machines, cross validation, and our own composite classification method (classifier voting) to produce results.

How does it work? We crawled the site CelebrityTypes to obtain our ground truth personality classifications. Then we collected quotes and tweets for the nearly 600 personalities we found. Our primary training set (8775 quotes) is a processed version of a crawled quote collection available from TheWebMiner. We used quotes that we crawled from Freebase as an evaluation set. Tweets were retrieved with Tweepy. Many of the names in our ground truth database matched WebMiner names and Twitter handles, but we had to match many of them by hand.

The text area on the homepage submits to a set of classifiers pre-trained on various versions of the primary training set, whose data and targets (ground truths) are available here and here, respectively. Any text submitted to Mech Jung is immediately transformed by various language processes and input into the appropriate "pipelines", which vectorize and classify it. A four-letter personality type is actually generated by four different classifiers, one for each preference (I/E, N/S, F/T, J/P). At the time of this writing, the committee of classifiers for the text submission consists of nine sets of four classifiers each. Each of the nine sets uses a different combination of classification and language processing method.
